# nowin_core
[![Build Status](https://travis-ci.org/g0v/nowin_core.svg)](https://travis-ci.org/g0v/nowin_core)

Nowin core library, this repo is for

 - Database schema (ORM)
 - Data model
 - Shared library

Most of other nowin projects depend on it.

# run tests

pip install -e .[tests]
